# Projet MVC - Guide d'installation rapide

## Installation de la base de données

1. Créez une base de données MySQL nommée `mvc_project`
2. Importez le fichier `database.sql` (à la racine du projet) dans votre gestionnaire de base de données.

## Lancement du projet

1. Placez les fichiers dans le dossier de WAMP/XAMPP 
2. Accédez à l'application via : `http://localhost/nom_du_dossier`
